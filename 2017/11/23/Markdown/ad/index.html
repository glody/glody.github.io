# DC info
user/password Administrator/ZStack-01
aduser zstest/ZStack-01
ip 172.20.190.228
域名 adtest.zs
DNS 172.20.190.228

#添加用户
1. 在DC（Domain Controller）上点击开始->Active Directory用户和计算机（172.20.190.228）
2. 点击左边栏的zstest（这是一个组织单位），也可以新建组织单位
3. 右键zstest -> 新建 -> 用户
4. 输入对应的用户信息点击下一步
5. 输入密码，要满足windows密码的复杂度，并勾选密码永不过期
6. 点击下一步直到成功

#添加主机
首先把172.20.190.228添加到网卡ipv4的DNS服务器中
1. 在对应的windows主机上点击我的电脑->属性
2. 在计算机名后面点击更改设置
3. 在计算机名页面点击更改
4. 选择隶属于域并填入adtest.zs
5. 更改计算机名并点击确定
6. 输入用户名密码zstest/ZStack-01
7. 重启windows主机
