Win10 installtion

  1. 添加镜像
    • 界面点击添加镜像，镜像类型iso，平台选择Windows，链接http://192.168.200.100/mirror/iso/win10.iso
    • 界面点击添加镜像，镜像类型iso，平台选择WindowsVirtio，链接http://192.168.200.100/mirror/iso/virtio-win-0.1-100.iso
  2. 创建vm
    • 镜像选择windows，点击控制台打开, 在硬盘分区页面出现的时候返回管理节点网页，卸载ISO，然后再挂载ISO，选择WindowsVirtio驱动并挂载。
    • 返回控制台，点击加载驱动程序->确定->浏览->CD驱动器->Win8->AMD64。去掉选择“隐藏与此计算机硬件不兼容的驱动程序”
    • 安装SCSI controller驱动和Ethernet Adapter驱动。
    • 返回管理节点网页，卸载ISO，然后再挂载ISO，选择Windows并挂载。
    • 返回控制台继续安装
